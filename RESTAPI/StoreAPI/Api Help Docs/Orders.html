This API is used to Get Customer Details on the basis of Customer Id

/api/orders/?pageIndex=1&pageSize=2&orderBy=Id&direction=asc

Parameter

Field Type Description
pageIndex int

Required page index
pageSize int

Required Page Size
orderBy string

Only "Id" or "Status" allowed to pass
direction string

Only "asc" or "desc" allowed to pass

Success 200

  • Response (example): 
    HTTP/ 200 Ok 
      [
    {
        "OrderId": 3,
        "TotalAmount": 750.0,
        "CustomerName": "Sample Name",
        "Email": "sample@gmail.com",
        "OrderStatus": "Ordered",
        "PaymentMode": "UPI"
    },
    {
        "OrderId": 4,
        "TotalAmount": 760.0,
        "CustomerName": "Sample Name",
        "Email": "sample@gmail.com",
        "OrderStatus": "Ordered",
        "PaymentMode": "UPI"
    }
]

Error 4xx

Field Description
Not Found

No Orders found for the given input.
  • Response (example): 
    HTTP/1.1 404 Bad Request
      "No orders found for the given input."
