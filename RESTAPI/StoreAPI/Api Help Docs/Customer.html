Get Customer

This API is used to Get Customer Details on the basis of Customer Id

/api/customer/{id}

Parameter

Field Type Description
Id int

Customer Id
  • Example usage: 
/api/customer/3

Success 200

  • Response (example): 
    HTTP/ 200 Ok 
    {
      
    "Id": 3,
    "FullName": "James Bond",
    "Email": "james.bond@gmail.com",
    "Mobile": "9000000007"
 
    }
Field Type Description
id String

The new Users-ID.

Error 4xx

Field Description
Not Found

No Customer found for the given input.
  • Response (example): 
    HTTP/1.1 404 Bad Request
      "No Customer found for the given input."
